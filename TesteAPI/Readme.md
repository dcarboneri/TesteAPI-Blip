Minha API utiliza C# pela segurança, velocidade e por eu ter maiores conhecimentos da linguagem, então é uma API .Net CORE onde eu subdidvido as funções em pasta para melhor organização.
O código é completamente baseado nos princípios de API Restful, princípios de clean code e SOLID e tambem possuí teste unitário utillizando xUnit.
Tambem usei outras libs no projeto como Newtonsoft.Json para lidar com o retorno da API do GitHub com mais praticidade e RestSharp para facilitar as requisições HTTP.
Todos os métodos possuem tratamento de exessão.
A API não foi publicamente publicada e integrada com o chatbot da Blip pois não possuo experiencia com deploy.
